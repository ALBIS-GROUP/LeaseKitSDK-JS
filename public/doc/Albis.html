Constructor

new Albis(settings)

Create an Albis object Note: due to security reasons, keep all sensitive data (i.e. APIid, APIsecret, username, ...)
Parameters:
Name Type Description
settings Object
Properties
Name Type Description
username string shop owner or shop admins Albis username
password string shop owner or shop admins Albis password
auth0Username string shop owner or shop admins auth0 username
auth0Password string shop owner or shop admin auth0 password
realm string shop owner connection name
provision number provision - defines how much commission, retailer wants to receives for each deal. Possible values min: 0, max: 5. Default 0. Value in half percentage i.e. 1.5 not 1.65
SDKendpoint string SDK endpoint
apiStage boolean defines proper API Gateway endpoints stage (API version) for requests
